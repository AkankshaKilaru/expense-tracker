# Expense Tracker Application

## Project Description
This project is a simple Expense Tracker application that helps users record, manage, and track their daily expenses. It allows users to monitor spending and maintain financial records efficiently.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add new expenses
- Display list of expenses
- Track total expenses
- Simple and user-friendly interface
- Responsive design for multiple devices

## How It Works
1. User enters expense details such as amount and description.
2. Expense is stored and displayed on the screen.
3. Application calculates and shows total expenses.

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Run the file:
   index.html
4. Application will open in browser.
